1.1 Notation

  • N : Z+Z^+Z+
  • N : n-bit positive integer
  • p : Prime (q always too)

1.2 Great Common Divisor

Fact : For all ints. x,y there exist ints a,b such that
a∗x+b∗y=gcd(x,y)a*x+b*y=gcd(x,y)a∗x+b∗y=gcd(x,y)

e.g. : gcd(12,18)=6=2∗12−1∗18gcd(12,18) = 6 = 2*12-1*18gcd(12,18)=6=2∗12−1∗18

1.3 Modular inversion

Lemma : x in ZN\mathbb{Z}_NZN​ has an inverse iff gcd(x,N) = 1; (ProofInPad)

Def : ZN∗\mathbb{Z}_N^*ZN∗​ = (invertible elements in ZN\mathbb{Z}_NZN​) = {x∈ZN:gcd(x,N)=1x\in\mathbb{Z}_N : gcd(x,N) = 1x∈ZN​:gcd(x,N)=1}
For Prime p, Zp∗\mathbb{Z}_p^*Zp∗​ = Zp\mathbb{Z}_pZp​/{0}={1,2,3,...,p-1}

1.3 Solving modular linear equations 解模线性同余方程

  • Solve : a∗x+b=0a*x+b=0a∗x+b=0 in ZN\mathbb{Z}_NZN​

2.1 费马小定理 (之后推广为Euler定理)

Thm : Let p be a prime, ∀x∈Zp∗:xp−1=1  in  Zp\forall x\in\mathbb{Z}_p^* : x^{p-1} = 1\;in\;\mathbb{Z}_p∀x∈Zp∗​:xp−1=1inZp​

e.g. :   p=5,35−1=81=1in  Z5p=5, 3^{5-1}=81=1\quad in\;\mathbb{Z}_5p=5,35−1=81=1inZ5​

推论 ：for x∈Zp∗⇒x∗xp−2=1⇒x−1=xp−2in  Zpx\in \mathbb{Z}_p^*\Rightarrow x*x^{p-2} = 1\Rightarrow x^{-1}=x^{p-2}\quad in\;\mathbb{Z}_px∈Zp∗​⇒x∗xp−2=1⇒x−1=xp−2inZp​

2.2 The structure of Zp∗\mathbb{Z}_p^*Zp∗​

Thm(Euler) : Zp∗\mathbb{Z}_p^*Zp∗​ is a cyclic group that is ∃g∈Zp∗\exist g \in \mathbb{Z}_p^*∃g∈Zp∗​ such that {1,g,g2,g3,...,gp−2}1,g,g^2,g^3,...,g^{p-2}\}1,g,g2,g3,...,gp−2} = Zp∗\mathbb{Z}_p^*Zp∗​
g is called generator (生成元) of Zp∗\mathbb{Z}_p^*Zp∗​

e.g. : p=7, {1,3,32,33,34,351,3,3^2,3^3,3^4,3^51,3,32,33,34,35}={1,3,2,6,4,5}=Z7∗\mathbb{Z}_7^*Z7∗​ 即 3 是 Z7∗\mathbb{Z}_7^*Z7∗​的一个生成元

For g∈Zp∗g\in \mathbb{Z}_p^*g∈Zp∗​ the set {1,3,32,33,...1,3,3^2,3^3,...1,3,32,33,...} is called the group generated by g (g的生成群)，denoted <g>
The Order of g ∈Zp∗\in \mathbb{Z}_p^*∈Zp∗​ is the size of <g>
Ordp(g)Ord_p(g)Ordp​(g)=|<g>| = smallest a>0 s.t. gag^aga = 1 in Zp\mathbb{Z}_pZp​

  • g在 Zp\mathbb{Z}_pZp​中的阶即g在 Zp\mathbb{Z}_pZp​中生成群的大小

  • 集合{1,g,g2,g3,...,gOrdp(g)−11,g,g^2,g^3,...,g^{Ord_p (g)-1}1,g,g2,g3,...,gOrdp​(g)−1}为g在 Zp\mathbb{Z}_pZp​中的生成群，其大小为g的阶，即Ordp(g)Ord_p (g)Ordp​(g)

  • 其中 最后一个元素 gOrdp(g)−1g^{Ord_p (g)-1}gOrdp​(g)−1 即Order的定义最后一句提到的 Zp\mathbb{Z}_pZp​中第一个满足ga=1g^a = 1ga=1 的数 a即为 Ordp(g)−1{Ord_p (g)-1}Ordp​(g)−1

  • Att. 费马小定理的幂a=p-1并不是第一个使得 ga=1g^a = 1ga=1 in Zp\mathbb{Z}_pZp​的a
    如 ${Ord_7 (2)} = 3 $ 而不是p-1的6

Thm.(Lagrange) : ∀g∃\forall g \exist∀g∃ in Zp∗\mathbb{Z}_p^*Zp∗​: Ordp(g)Ord_p (g)Ordp​(g) divides p-1

Euler's gemeralization of Fermat (由费马小定理推出的欧拉定理) Def.
欧拉方程ϕ\phiϕ Def. For an integer N define ϕ(N)=∣ZN∗∣\phi(N) = |\mathbb{Z}_N^*|ϕ(N)=∣ZN∗​∣

欧拉定理 Def. ∀x∈ZN∗\forall x \in \mathbb{Z}_N^*∀x∈ZN∗​ : xϕ(N)=1x^{\phi (N)} = 1xϕ(N)=1 in ZN\mathbb{Z}_NZN​

e.g.

  • $\phi (12) = $|{1,5,7,11}| = 4;
  • $\phi (p) = p-1 $
  • 对于 半质数 N = p*q (p,q均是prime)
    ⇒\Rightarrow⇒ ϕ(N)=N−p−q+1=(p−1)(q−1)\phi(N) = N-p-q+1 = (p-1)(q-1)ϕ(N)=N−p−q+1=(p−1)(q−1)

  • 5ϕ(12)=54=625=15^{\phi(12)} = 5^4 = 625 = 15ϕ(12)=54=625=1 in ZN\mathbb{Z}_NZN​
