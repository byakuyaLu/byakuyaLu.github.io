Solution for question 4

Suppose you are told that the one time pad encryption of the message "attack at dawn" is 6c73d5240a948c86981bc294814d (the plaintext letters are encoded as 8-bit ASCII and the given ciphertext is written in hex).
What would be the one time pad encryption of the message "attack at dusk" under the same OTP key?

ascii_message1 = "attack at dawn"
ascii_message2 = "attack at dusk"
hex_cipher1 = "6c73d5240a948c86981bc294814d"

def strxor(a, b):    
    if len(a) > len(b):
       return "".join([chr(ord(x) ^ ord(y)) for (x, y) in zip(a[:len(b)], b)])
    else:
       return "".join([chr(ord(x) ^ ord(y)) for (x, y) in zip(a, b[:len(a)])])

ascii_cipher1 = ""
for i in range(0,len(hex_cipher1),2):
	ascii_cipher1 += chr(int(hex_cipher1[i]+hex_cipher1[i+1],16))

ascii_key = strxor(ascii_message1,ascii_cipher1)

ascii_cipher2 = strxor(ascii_message2,ascii_key)

hex_key = ""
for i in ascii_key:
   hex_key += (hex(ord(i))[2:].zfill(2))
print(hex_key)

hex_cipher2 = ""
for i in ascii_cipher2:
	hex_cipher2 += (hex(ord(i))[2:].zfill(2))  
print(hex_cipher2)

The outcome of the program is

0d07a14569fface7ec3ba6f5f623
6c73d5240a948c86981bc2808548

So the one time pad encryption of the message "attack at dusk" under the same OTP key is

6c73d5240a948c86981bc2808548
