The Birthday Paradox:

     Let r1,r2,r3,...,rn∈Ur_1,r_2,r_3,...,r_n \in Ur1​,r2​,r3​,...,rn​∈U be independent identically distributed

When n = 1.2∗∣U∣1/21.2*|U|^{1/2}1.2∗∣U∣1/2 then Pr[∃ i≠j:ri=rj]≥12i \neq j:r_i=r_j]\ge \frac{1}{2}i​=j:ri​=rj​]≥21​

Method:

To achieve the result above, I designed the programm as below.

  • The collection of all integers between [0,2322^{32}232] format the UUU
  • Considering the time complexity of this experiment would be huge, I used C++ to generate the data and analyzed the data by python.
  • The distribution of the elements in UUU must be uniform while the rand() function of C++ can't generate uniform distributed elements even if I change the seed everytime. Therefore, I used the new method in C++ 11 to generate the numbers, whose outcome is guaranteed to be uniformlly distributed . The method is demonstrated as below.

Souce code:

#include <bits/stdc++.h>
#define ll long long

using namespace std;

unordered_map<ll, bool> mp;
const ll Rand_max = pow(2, 32);     // define the scope of the numbers
// define the Repetion times of every sample experiment
// It's obvious that the Repeat_times larger, the probability more accurate
const int Repeat_times = 1000;     
// define the max samples
const ll tMax = Rand_max;           

int main() {
    random_device rd;
    int collision_times;
    freopen("output.txt", "wt+", stdout);

    // define the function to calculate the probablity
    auto ProbabilityFunction = [&collision_times,&rd](const int NumOfSamples) -> void {
        mt19937 rng(rd());                                  
        uniform_int_distribution<ll> uni(0, Rand_max);      // generator random numbers under uniform distribution
        for (int i = 0; i < Repeat_times; i++) {
            mp.clear();
            for (int j = 0; j < NumOfSamples; j++) {
                ll tmp = uni(rng);
                if (mp[tmp]) {collision_times++;break;} 
                else mp[tmp] = 1;
            }
        }
        float prob = collision_times / (float)Repeat_times;
        printf("%f \n",prob);
    };
//  calculate the probability of collision of sample t samples from [0,32]
//  Sample 1 point per 500 points
    for (ll t = 500; t <= tMax;t+=500) {  
        mp.clear();
        collision_times = 0;
        ProbabilityFunction(t);
    }
}

Conclusion of the programm

  • The probability ended manually when the probablity of collision is bigger than 0.99 continuously.
  • To achieve that result, the programm have running for approximately 4 hours and the variable t reached 234,000.
  • When the t reached approximately 76,500, the probability exceeded 0.5 for the first time, which is basiclly corresponding with The Birthday Paradox
